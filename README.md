# Sobre mim

Sou um **Desenvolvedor Full Stack** apaixonado por transformar ideias em soluções digitais modernas, intuitivas e escaláveis.

Atualmente concentro meu trabalho no desenvolvimento de aplicações completas utilizando **Java**, **Spring Boot**, **React**, **TypeScript** e **PostgreSQL**, aplicando princípios como **Clean Architecture**, **REST APIs**, **autenticação segura** e boas práticas de engenharia de software.

Minha principal motivação é criar softwares que entreguem valor real aos usuários, combinando código limpo, arquitetura sólida e interfaces modernas.
